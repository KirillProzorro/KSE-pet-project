# Air Raid Alerts Time Series Analysis (Kyiv)

## Опис проєкту
Міні-проєкт для аналізу часових рядів повітряних тривог у Києві. Додаток автоматично збирає дані з публічного джерела (kyiv.digital), виконує очищення та співставлення циклів "початок-відбій", розраховує статистичні метрики та візуалізує їх.

## Архітектурні рішення
* **Стековий алгоритм парсингу:** Дані на джерелі подаються у зворотній хронології та можуть містити незавершені тривоги. Алгоритм використовує стек для надійного пошуку парних подій та відсікання аномалій.
* **In-memory кешування:** Для запобігання блокуванню (Rate Limiting) з боку донора, результати обчислень кешуються на 15 хвилин.
* **Локалізація часу:** Імплементовано зсув UTC+3 для коректного розрахунку щоденної статистики за київським часом.

## Стек технологій
* **Backend:** Python 3, Flask
* **Scraping & ETL:** Requests, BeautifulSoup4, re, datetime
* **Frontend:** HTML/CSS, Chart.js (візуалізація)

## Локальний запуск

1. Клонувати репозиторій:
   git clone [посилання_на_ваш_репозиторій]
   
2. Встановити залежності:
   pip install flask requests beautifulsoup4
   
3. Запустити сервер:
   python app.py
   
4. Відкрити в браузері:
   http://localhost:5000
